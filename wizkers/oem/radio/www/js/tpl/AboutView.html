About

Wizkers:Radio is a rig controller app for Amateur Radio equipment.

Version 0.2.2

  • New instrument: S-Level monitor
  • KX2 advanced settings
  • LCD Screen bargraph works on KX2 and KX3

Version 0.2.1

  • Support for Bluetooth serial dongles on Android ('classic' bluetooth SPP modules)
  • VFO Wheel working on Android as well (experimental)
  • VFO A Wheel on KX2 is now active

Version 0.2.0

  • New Radio: Support for the Elecraft KX2

Version 0.1.8

  • New radio: Yaesu FT817ND, preliminary support
  • New instrument: Elecraft KX3 as SWR sweeper
  • Large performance enhancements on XML-RPC support

Version 0.1.7

  • XG3: lots of improvements on usability and reliability.
  • KX3: Frequency cards scroll following VFO A

Version 0.1.6

  • KX3: Frequency card color follows VFO
  • XG3: Support for sweep functionality

Version 0.1.5

  • KX3: experimental memory viewer
  • KX3: experimental flash memory explorer
  • New XMLRPC output connector to work with fldigi

Version 0.1.3

  • Support for extended KX3 settings.
  • Command terminal in Device setup view for KX3.

Version 0.1.2

  • Support for Elecraft XG3.

Version 0.1.0

  • Initial release.
