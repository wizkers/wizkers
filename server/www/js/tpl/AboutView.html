Comments and questions

I would love to hear your feedback: ed@wizkers.io

0.10.5

  • Added geolocation capabilities for Onyx and Blue Onyx.
  • Android version: Blue Onyx implementation.
  • Android version: Safecast bGeigie support.
  • Android version: upload to Safecast for bGeigie.
  • Lots of bug fixes and improvements.

0.10.0

  • Live view graph is now restored when coming back to the home screen.
  • Serial port connections also support TCP sockets for serial-over-tcp dongles.
  • Elecraft KX3: Data mode macros are configurable

0.9.20

  • User experience: multiple useability improvements.

0.9.19

  • New Output plugin: WebRTC channel.
  • New instrument: Remote KX3 through WebRTC (experimental).

0.9.18

  • Support for Bluetooth Low energy (ChromeOS only)

0.9.17

  • Support for latest Onyx firmware (12.30)
  • KX3: support for audio input device selection
