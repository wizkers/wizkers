Help

You can test commands on the dongle here. Type "?" to get started..

Equivalent Frequency: this converts the current CPM value into a frequency in Hz, which will be different from Raw frequency unless the dead time factor is zero.

Raw Frequency: the raw frequency measured on input over the last 3 seconds.

Counter Windows: depending on the current CPM count, the counter will calculate the CPM using a sliding time window: the higher the CPM, the shorter the window. The dongle supports three windows that can be setup here. Window 3 duration is fixed.
